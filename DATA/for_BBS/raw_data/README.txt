for_BBS: raw data folder: 
1) BBSdata_accessed_03dec2020 folder contains birds' count data at 50 stop along each route (source: Pardieck, K.L., Ziolkowski Jr., D.J., Lutmerding, M., Aponte, V.I., and Hudson, M-A.R., 2020, 
North American Breeding Bird Survey Dataset 1966 - 2019: U.S. Geological Survey data release, https://doi.org/10.5066/P9J6QUF6)
2) EltonTraits folder contains birds' diet category info as BirdFunDat.txt file (source: Hamish Wilman, Jonathan Belmaker, Jennifer Simpson, Carolina de la Rosa, Marcelo M. Rivadeneira, Walter Jetz. 2014. EltonTraits 1.0: Species-level foraging attributes of the world's birds and mammals. 
Ecology 95:2027. http://dx.doi.org/10.1890/13-1917.1)
